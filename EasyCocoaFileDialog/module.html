EasyCocoaFileDialog¶

Introduction¶

This module provides an easy abstraction of NSOpenPanel and NSSavePanel for Python script writers that would like to prompt the user for input/output files without the ugliness of using the command line.

The module requires the Cocoa module, which is part of PyObjC.

Tip

PyObjC appears to be included with OS X, at least as of 10.10 Yosemite.

Example¶

prompt = "Please pick the input data file."
filename = chooseFileToOpen(prompt)
if filename is None:
   print "The user clicked Cancel."
else:
   print "The user picked: %s" % filename

Reference¶

EasyCocoaFileDialog.chooseFileToOpen(prompt="Select the file to open")¶

Show a Cocoa NSOpenPanel prompting the user for the path of a single file to open.

Parameters:prompt (str) – The message shown at the top of the NSOpenPanel
Returns:the path to the file chosen by the user, or None if the user clicks Cancel.
EasyCocoaFileDialog.chooseFileToSave(prompt="Select the file to save")¶

Show a Cocoa NSSavePanel prompting the user for a path of a file to save to.

Parameters:prompt (str) – The message shown at the top of the NSSavePanel
Returns:the path to the file chosen by the user, or None if the user clicks Cancel.
